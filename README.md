# SvelteKit Dashboard

Admin dashboard with collapsible sidebar, live stats cards, Chart.js revenue chart, and searchable user table.

**Stack:** Sveltekit | **Complexity:** moderate
**Tags:** `sveltekit`, `svelte`, `dashboard`, `admin`, `charts`, `tables`, `typescript`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
